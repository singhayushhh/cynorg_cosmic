# cynorg_cosmic
A futuristic 3D interactive website inspired by IIT Bombay Techfest, built using HTML, CSS, JavaScript, and modern web animations.
Techfest IIT Bombay – 3D Website

Overview

This project is a 3D animated website concept created for Techfest IIT Bombay. It is designed to provide an immersive and futuristic experience that reflects the spirit of innovation and technology associated with Techfest.

Inspiration

The website draws inspiration from the curiosity, creativity, and technological advancements showcased by Techfest IIT Bombay. Its space and cosmic theme symbolizes exploration, discovery, and the limitless possibilities of science and engineering.

Features

- Futuristic space-inspired UI
- Interactive 3D objects
- Smooth 3D scroll-based animations
- Animated page transitions
- Responsive and modern design
- Sections highlighting Techfest and its vision
- Immersive user experience with dynamic visuals

Technologies Used

- HTML5
- CSS3
- JavaScript
- Three.js
- GSAP

Project Vision

The goal of this project is to reimagine the digital experience of Techfest IIT Bombay through interactive 3D elements and modern web technologies. By combining motion, depth, and storytelling, the website aims to capture the excitement and innovative spirit of one of India's premier science and technology festivals.

Conclusion

This project demonstrates how creative design and 3D web technologies can be used to create engaging and memorable experiences. The cosmic theme and interactive elements are intended to inspire curiosity and reflect the futuristic essence of Techfest IIT Bombay.
